# LimitBar

Codex と Claude Code の利用状況を確認できる
macOSメニューバーアプリです。

メニューバーから現在の利用率を確認でき、
設定したしきい値に近づいたときやリセット時に通知します。

## 主な機能

- macOSメニューバー常駐アプリ
- Dockに表示されない軽量UI
- メニューバーから利用状況を確認
- フローティングウィジェット表示
- 自動更新 / 手動更新
- 利用率しきい値通知（例: 90%）
- リセット通知
- 設定の永続保存（UserDefaults）

##必要環境

- macOS 14 以降（推奨）
