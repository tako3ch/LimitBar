Features
Everything you need
Real-time monitoring
Fetches Codex and Claude Code usage automatically at your chosen interval.
Floating widget
Detachable floating widget with configurable size, position, opacity, and layer order.
Smart notifications
Get notified when usage approaches your threshold or resets.
Usage history
Visual usage report with history graphs to track consumption over time.
Session auto-detect
Connects via Codex app or CLI auth, Claude in-app login, browser sessions, or Claude desktop app. No API keys needed.
Bilingual
Automatically switches between English and Japanese based on system language.
